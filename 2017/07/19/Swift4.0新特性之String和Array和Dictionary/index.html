字符串索引

  • Swift 的字符串不能通过整数值索引
  • 使用 index(before:) 和 index(after:) 方法来访问给定索引的前后。要访问给定索引更远的索引，你可以使用 index(_:offsetBy:) 方法而不是多次调用这两个方法。
  • Swift 4添加了Substring类型，用于引用String的子序列。
